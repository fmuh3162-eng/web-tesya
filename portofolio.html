Tentang Saya

Saya Tesya Indriani, seorang Make Up Artist (MUA) sekaligus lulusan SMK Negeri 1 Majene, jurusan Tata Kecantikan, tahun 2024. Sejak masa sekolah di SMK, saya memiliki minat besar di dunia kecantikan dan seni merias wajah. Saya mempelajari berbagai teknik make up — mulai dari rias sehari-hari, make up pesta, hingga make up pengantin dan karakter. Dari pengalaman itu, saya semakin yakin untuk menekuni dunia kecantikan sebagai karier profesional. Saat ini, saya melanjutkan pendidikan di Universitas Sulawesi Barat, program studi Pendidikan Teknologi Informasi. Walau fokus kuliah saya di bidang teknologi, saya tetap aktif mengembangkan keterampilan make up dan menerima berbagai proyek rias, seperti wedding, wisuda, photoshoot, hingga acara formal. Bagi saya, make up bukan sekadar mempercantik tampilan, tetapi juga cara untuk membantu seseorang tampil lebih percaya diri dan bahagia dengan dirinya sendiri.💄
